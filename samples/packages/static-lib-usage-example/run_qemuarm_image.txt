2. The Cross-Compiled C file binary should be located in the directory on which it is installed in the Recipe file along with the specified file permissions.
root@qemuarm:~# ls -l /usr/bin/program
-rwxr-xr-x    1 root     root          5508 Mar  9  2018 /usr/bin/program

3. The Cross-Compiled C file binary should be executed and it should use the functions specified by the static library.
root@qemuarm:~# program
Hello World!
Hello World!
Hello World!
Hello World!
Hello World!

4. However, the static library or the Header file must not be present in the Image. The static library is already a part of the 'program' binary.
root@qemuarm:~# ls -l /usr/lib/lib_static.a
ls: /usr/lib/lib_static.a: No such file or directory
root@qemuarm:~# ls -l /usr/include/static_lib.h
ls: /usr/include/static_lib.h: No such file or directory

5. To exit the QEMU Console, run the below command.
root@qemuarm:~# poweroff
